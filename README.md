# Big Bites React Conversion

Production-ready React + Vite + Tailwind CSS v4 project converted from the original HTML/CSS/JS website.

## Stack

- React 19.2
- Vite 8
- Tailwind CSS 4.2 via `@tailwindcss/vite`
- React Router DOM 7
- Lucide React icons
- React Hot Toast notifications

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Project structure

```text
src/
├── components/
├── data/
├── hooks/
├── pages/
├── sections/
├── App.jsx
├── main.jsx
└── styles.css
```
